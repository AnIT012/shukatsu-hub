# 就活ダッシュボード

毎朝開いて「**次に何をすべきか・次の締切はいつか**」が一目でわかる、自分専用の就活インターン進捗管理アプリ。データはブラウザの `localStorage` に保存され、サーバー・DBは不要です。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開く。

## 使い方

1. 右上「**企業を追加**」で最初の企業を登録（初期データは空）。
2. カードをクリックすると右からパネルが開く。そこで **選考ステップ**（エントリー→ES→Webテスト→面接…）と締切、関連リンク、メモを編集。
3. 各ステップの状態は左の丸（●）を**ワンタップ**で `未着手 → 進行中 → 完了` と切替。完了にすると、自動的に次の未完了ステップが「次のアクション」になる。
4. 一覧は「次のアクション＋締切」が主役。締切3日以内＝赤、7日以内＝黄でハイライト。
5. 編集はすべて**自動保存**（ヘッダーに「保存しました」表示）。

## バックアップ / 端末移行

- ヘッダーの ↓（エクスポート）で全データを JSON 保存。
- ↑（インポート）で JSON を読み込み（現在のデータを置き換え）。

## 技術スタック

Next.js (App Router) / React / TypeScript / Tailwind CSS / shadcn/ui スタイル / lucide-react / next-themes（ダークモード）/ sonner（トースト）。

## データ構造（要点）

`src/lib/types.ts` 参照。1社 = `Application` が複数の `SelectionStep`（種別・締切・状態・メモ）を持ち、「次のアクション」は `src/lib/next-action.ts` が *未完了で最も締切が近いステップ* を自動算出します。
